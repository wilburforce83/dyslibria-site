Installation

Install our software by following these steps:


# Install Node.js
curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
sudo apt-get install -y nodejs

# Clone and run
git clone https://github.com/yourgithubrepo.git
cd yourrepo
npm install
npm start
